# Tensor Train Decomposition Project

## Description

This project implements several algorithms related to **Tensor Train (TT) decomposition**.

The project follows a `src` layout.  
All core algorithms are implemented inside:

```text
src/tt_project/
```

---

# Project Structure

## Core Algorithms

- `tt_svd.py` – TT-SVD algorithm with fixed ranks  
- `tt_svd_delta.py` – TT-SVD algorithm with adaptive ranks (based on a tolerance δ)  
- `rounding.py` – Tensor rounding procedure  
- `tt_rounding.py` – Alternative implementation of the rounding algorithm  
- `tt_partialcontractionRL.py` – Right-to-left partial contraction  
- `left_ortho.py` – Left orthogonalization  
- `right_ortho.py` – Right orthogonalization  

---

## Tensor Operations

- `tt_tensor.py` – Reconstruction of the full tensor from TT cores  
- `sum_tensor.py` – Sum of two tensors in TT format  
- `square_root_sum_tensor.py` – Example tensor used for testing  
- `delta_rank.py` – Compute minimal TT ranks for a prescribed error tolerance δ  

---

## Utilities

- `v2h.py` – Convert a vertical unfolding into a horizontal unfolding  
- `horizontal_core.py` – Reshape a 3D tensor into its horizontal unfolding  
- `err.py` – Compute the error between the full tensor and its TT approximation  

---

## Test Scripts

- `test_tt_svd_delta.py` – Study the approximation error on two example tensors  
- `test_tt_rounding.py` – Study the error and rank variation for TT rounding  
- `test_tt_pttk.py` – Test the implementation of the PTTK algorithm  

All figures generated by the test scripts are saved in:

```text
tests/figures/
```

# Installation

The project was developed using Python 3.11.

Install the required packages:

```bash
pip install numpy scipy matplotlib scikit-learn pytest
```

Install the project in editable mode:

```bash
pip install -e .
```

---

# How to Run

Run a specific test script from the project root directory.

Example:

```bash
python tests/test_tt_svd_delta.py
```

or:

```bash
python tests/test_tt_rounding.py
```
